# GitHub Pages Yayın Notu

Bu klasör, `Salise Ligi` gizlilik politikası sayfasını GitHub Pages üzerinden yayınlamak için kullanılır.

## GitHub Pages nasıl açılır?

1. GitHub üzerinde repo sayfasına girin.
2. `Settings` > `Pages` menüsünü açın.
3. `Source` bölümünde `Deploy from a branch` seçin.
4. Branch olarak `main`, klasör olarak `/docs` seçin.
5. Kaydedin ve yayınlama tamamlanana kadar birkaç dakika bekleyin.

## Yayınlanan URL örneği

- Format: `https://<kullanici-adi>.github.io/<repo-adi>/`
- Örnek: `https://ornek-kullanici.github.io/salse-ligi-cursor-v3/`

> Not: Gizlilik politikası sayfası `docs/index.html` olduğu için yayınlanan URL doğrudan bu sayfayı açar.
